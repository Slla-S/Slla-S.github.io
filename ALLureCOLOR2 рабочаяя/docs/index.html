Как использовать генератор палитр

1. Нажмите кнопку "Сгенерировать палитру" для создания случайных цветов

2. Заблокируйте понравившиеся цвета, чтобы они не менялись при следующей генерации

3. Сохраните палитру, чтобы использовать её в своих проектах

4. Просматривайте и управляйте сохраненными палитрами во вкладке "Мои палитры"

Типы палитр

Пастельные тона: Мягкие, приглушенные оттенки

Монохромная палитра: Оттенки одного цвета
